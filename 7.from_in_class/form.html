FORM :

->The form tag is one of the most important tag of html used for sending data from client to server

->It is used for creating user interface which takes data from user and sends to the server

->The form tag hosts another tag called as input which is used for creating windows element (controls) such as textbox,checkbox ,button , list etc. .

Attributes of form tag

1.action="server script"

2.Method="GET" , "POST"

"GET" : The get method is used for sending limited adn plain data(text data) .By get the data send to the server appears in the address bar means all the data which is being send will be visible to the user. it means the get is safe for sending secure data such as password , otp, card card details,

"POST" :
- Used for sending large data
- Binary files such as image , pdfs, etc. can also be send
- The post method doesn't show the query string in the address bar , so it is safe

INPUT TAG :

Attributes fo input tag :

1.Type : text/password/button...etc.

2.id : Any id (used by js)

3.Name : Any name(used by js and, server side script)
