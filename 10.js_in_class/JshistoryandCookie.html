History Object

~ History object is a part of browser which stores addresses of all the the visited pages in stack format. we can use backward and forward button to navigate all the pages.

~ In the case of transactional website the history object the breaks the security rules , So we should handle it using javascript So that the pages cannot go back.

Cookie Object

~ Cookie is a text file created by a website on the client computer . Normally it is plain text file but can be encrypted also .

~ It is used for tracking some information like location, date/time, page visit, username, password etc.
